## 简化版结构

```text
输入
  ↓
[ 共享时间尺度建模 ]
   ↙             ↘
[ 文本构造支路 ]   [ 控制支路 ]
   ↘             ↙
     [ 条件扩散主干 ]
             ↓
         [ 最终预测 ]
```

## 更偏图形化的表达

```text
历史序列 + 时间信息 + 文本来源
                ↓
        [ 时间尺度透镜 / Scale Lens ]
             ↙               ↘
   [ 动态文本构造 ]       [ 训练与推理控制 ]
             ↘               ↙
          [ 条件扩散预测主干 ]
                    ↓
               [ 未来预测 ]
```

## 读图说明

- 中间是“共享时间尺度建模层”，它是整张图的核心
- 上支路是“文本构造支路”，负责生成高质量文本条件
- 下支路是“控制支路”，负责多分辨率训练和自适应 guidance
- `pi_i` 是两条支路之间的桥
- 两条支路最终都汇入“条件扩散主干”，共同决定最终预测结果

## 一句话总结

```text
时间尺度表征既决定“模型看什么文本”，也决定“模型如何训练、推理时多大程度相信文本”，最终两条路径共同作用于条件扩散预测。
```
